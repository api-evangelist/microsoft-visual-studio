Visual Studio now supports Agent Skills , which are reusable instruction sets that teach Copilot agents how to handle specific tasks like running a build pipeline, generating boilerplate, or following your team’s coding standards. Define a skill once, and the agent applies it automatically whenever it’s relevant. Creating a skill You can create a skill directly from within Visual Studio.
